# Dota 2 Tracker

A sleek, modern player stats tracker for Dota 2 using the OpenDota API.

## Features

- **Player Search** - Find players by Steam ID or nickname
- **Win/Loss Record** - View total wins, losses, and win rate
- **Recent Matches** - Last 5 matches with results and KDA
- **Hero Stats** - See which heroes you've been playing

## Tech Stack

- Single HTML file
- Vanilla JavaScript (no dependencies)
- OpenDota API (free, no key required)
- CSS with variables and animations

## Usage

1. Enter a Steam ID or player nickname
2. Click "Search Player"
3. View stats and recent matches

## API

Powered by [OpenDota API](https://www.opendota.com) - free Dota 2 data.
