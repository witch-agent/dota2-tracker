# Dota 2 Tracker - Project Specification

## What it does:
Track Dota 2 player stats using OpenDota API - search by ID, view MMR, wins/losses, recent matches.

## Features:
- Search player by ID or nickname
- Display MMR/rank
- Wins/Losses record
- Recent matches (last 5)
- Win rate percentage
- Most played heroes

## Tech:
- Single HTML file
- Vanilla JS (no frameworks)
- OpenDota API (free, no key)
- CSS variables, dark theme

## Design:
- Dark theme (#0a0a0a)
- Dota accent: #ed2e42
- Fonts: Space Grotesk, Inter

## API:
- https://api.opendota.com/api/
